✦ FAQ
Dúvidas frequentes
Preciso de cartão para o trial? +
Não. O trial de 7 dias é 100% grátis, sem cartão. Você só informa o pagamento quando decidir continuar.
Como funciona o widget embeddável? +
É uma tag <script> de uma linha. Cole no HTML do seu produto e o FeedbackFlow aparece como um botão flutuante para seus usuários.
Posso cancelar a qualquer momento? +
Sim, sem burocracia. Cancele com um clique dentro da plataforma. Você não é cobrado no próximo ciclo.
Quantos usuários podem votar no roadmap? +
Ilimitado. Seus usuários finais acessam o roadmap e votam gratuitamente — você paga apenas pela sua conta de gestor.
Tem API para integrar com meu sistema? +
Sim, nos planos Trimestral e Semestral você tem acesso à API REST completa com documentação detalhada.
